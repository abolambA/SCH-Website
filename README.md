# SpeedCubeHub Shopify Theme

This is the custom Shopify theme for SpeedCubeHub, designed for speed and optimal user experience.

## Features
- **Responsive Design:** Fully tailored for both Desktop and Mobile experiences.
- **Dynamic Mega Menu:** Curated "Shop by Brand" and "Shop by Type" navigation.
- **Interactive Hero Section:** 3D Tilt effects and smooth slider.
- **Advanced Search Dialog:** Quick access to products.
- **Bilingual Support:** Seamless toggling between English (EN) and Arabic (AR).

## Local Development
To run this theme locally, you need [Shopify CLI](https://shopify.dev/docs/themes/tools/cli) installed.
1. Run `shopify theme dev` in the terminal within this directory.
2. Follow the prompt to log in and preview the localized storefront.

## Deployment
This repository tracks the theme's codebase. Push changes to the `main` branch to sync with the live or development themes using Shopify GitHub integration, or use `shopify theme push`.
